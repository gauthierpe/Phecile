Phé-cile 🩵 — la Phé facile !

Interface webapp pour calculer les parts de phénylalanine (PCU) — France

🌐 Webapp en ligne : phecile.fr

C'est quoi Phé-cile ?
Phé-cile est une interface webapp gratuite, faite maison par une maman d'enfant PCU, pour aider au quotidien les familles et les personnes atteintes de phénylcétonurie (PCU) à calculer facilement leurs parts de phénylalanine.
Elle fonctionne sans installation, directement dans le navigateur, et peut être ajoutée à l'écran d'accueil du smartphone comme une vraie application (iOS et Android).



Fonctionnalités

📋 Liste d'aliments mémorisés : ajoutez vos aliments habituels une fois, retrouvez-les à chaque utilisation avec un curseur pour ajuster le poids en temps réel

🟡 Mode Liste : choisissez un fruit ou légume nature dans la liste intégrée (160+ aliments)

🟢 Mode Disque vert (Phetout) : produits transformés sans lait végétal ni céréales — entrez le taux de protéines de l'étiquette

🔴 Mode Disque rouge (Phetout) : pomme de terre seule ou produits avec lait, lait végétal ou céréales — entrez le taux de protéines de l'étiquette

💾 Mémoire locale : vos aliments sont sauvegardés sur votre appareil (localStorage), aucune donnée envoyée sur internet

📱 Progressive Web App : installable sur iPhone et Android directement depuis le navigateur



Sources des données

Cette webapp n'a pas d'intelligence propre, ni de puissance de calcul. Elle permet simplement un affichage plus sympa des tableaux d'équivalence (excel ou pdf) fournis par les équipes de suivi. 

- Consensus national SFEIM 2009 — Diététiciens SFEIM (Société Française des Erreurs Innées du Métabolisme) : https://filiere-g2m.fr/education-therapeutique-du-patient/outils-pedagogiques/liste-des-parts-de-proteines-et-acides-amines
- Tableaux du Phetout — phenylcetonurie.org : https://phenylcetonurie.org/WordPress3/wp-content/uploads/2025/08/PHETOUT-multiple-pour-parents-2.pdf



⚠️ Application non médicale — toujours vérifier avec votre diététicien(ne) spécialisé(e).
